# Toy CRM

Uma aplicação simples, escrita em COBOL, para praticar os conceitos do curso [Cobol: desenvolvendo um CRM com arquivos indexados](https://www.alura.com.br/curso-online-cobol-desenvolvendo-crm-arquivos-indexados) da Alura.

Para rodar o programa, primeiramente compile-o com:

```sh
./build.sh
```

Depois execute o programa:

```sh
./crm.bin
```
